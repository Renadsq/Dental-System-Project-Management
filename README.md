# Dental System – Software Project Management Plan

This project was developed as part of **SWE 387 – Software Project Management**
at King Fahd University of Petroleum & Minerals (KFUPM).

## Project Overview
The project focuses on developing a **Software Project Management Plan**
for a Dental System serving the KFUPM community, following the **IEEE SPMP standard**.

## Project Deliverables
- Software Project Management Plan 
- Project Schedule, Cost, and Budget Estimation
- Risk Management and Resource Planning
